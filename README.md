# MCP Test

이 저장소는 MCP(Model Context Protocol) 테스트를 위한 프로젝트입니다.

## 소개
이 프로젝트는 GitHub API를 활용한 테스트 저장소입니다.

## 기능
- GitHub API 통합
- 파일 관리
- 저장소 관리

## 시작하기
이 저장소를 클론하려면:
```bash
git clone https://github.com/Daniel-E-Son/MCP-Test.git
```

## 기여하기
프로젝트에 기여하고 싶으시다면 Pull Request를 보내주세요.
